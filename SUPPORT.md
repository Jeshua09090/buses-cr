# Support

Buses CR is still early, so support is mostly project discussion and issue
triage.

Use GitHub Issues for:

- bugs in the app
- route or stop data corrections
- unclear setup docs
- planner results that do not match real trips

Please include enough detail for someone else to reproduce or verify the issue.
For route-planning reports, origin, destination, approximate time, expected
route, and actual route are the most useful details.

For private security concerns, use GitHub Security Advisories when possible.
